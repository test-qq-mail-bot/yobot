# 使用酷Q安装

::: tip

开始使用前  
如果提供服务的QQ号是刚注册的、长期未使用的、异地登录的，则很容易被封禁。  
建议在服务器上将账号**挂机一周**再提供服务

yobot搭建难度约等于搭建网站的难度，请量力而行

:::
